Body:   
 
A set of parameters defining the configuration for a particular collection of notes. This object supplies a note collection type, a record definition, and a preferred file extension. 

Notable methods:

* getWidgetWithLabel -- Creates a label and a data entry widget for the passed DataFieldDefinition.
